 Introduction:

- Welcome to the Brick Breaker game! This simple game is built using HTML, CSS, and JavaScript. 
- This is a classic game where you control a paddle to bounce a ball and break bricks. 
- The goal is to break all the bricks on the screen to win the game and to score points.


 How to Play:

- Press the spacebar to start the game.
- Use the left and right arrow keys or "A" and "D" keys to move the paddle.
- The ball will bounce off the walls and paddle to break the bricks.
- Your score increases as you break bricks.
- If the ball falls below the paddle, the game ends.
- Break all the bricks to win the game.


 Rules To Keep in Mind:

- The game starts when you press the spacebar.
- Move the paddle to bounce the ball and break the bricks.
- If the ball passes the paddle and reaches the bottom, the game is over.
- You win by breaking all the bricks.
- You lose if the ball passes the paddle and reaches the bottom.
- Your final score is displayed when the game is over.
